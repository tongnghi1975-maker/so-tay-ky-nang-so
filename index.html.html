📱 Hướng dẫn mở app trên điện thoại (iPhone & Android)

🍎 Trên iPhone / iPad

  1. Mở app Tệp (Files) → tìm tới file So_tay_Ky_nang_so…html (thường ở mục Tải xuống / Downloads hoặc Trên iPhone của tôi).
  2. Chạm giữ vào file → chọn Chia sẻ (Share).
  3. Trong bảng chia sẻ, chọn Safari (hoặc Chrome). App sẽ mở đầy đủ trong trình duyệt.

🤖 Trên Android

  1. Mở app Quản lý tệp (File Manager / Files / My Files) → vào thư mục Download / Tải xuống.
  2. Chạm giữ (hoặc nhấn) vào file So_tay_Ky_nang_so…html → chọn Mở bằng (Open with).
  3. Chọn Chrome hoặc Cốc Cốc (đừng chọn trình xem mặc định của máy).
💡 Mẹo: Nếu nhận file qua Zalo / Messenger / Gmail, hãy tải file về máy trước rồi mở lại bằng Chrome/Safari — không mở trực tiếp trong các app đó (app có thể không bấm được nút hoặc không lưu được tiến độ). Trên iPhone, tránh chế độ "Xem nhanh (Quick Look)".

Tiến độ học tập của bạn được lưu ngay trên máy/trình duyệt này. Sau khi hoàn thành bài kiểm tra, hãy tải file kết quả và gửi cho Phòng Công nghệ thông tin để được xác nhận hoàn thành chính thức.
